# Trenni

Trenni is an open-source, cross-platform budget management application for iOS, Android, Windows and macOS written in Flutter. Unlike traditional personal finance applications which often only track past expenses, Trenni focuses on planning the future using the sinking funds method. It is local-first with optional peer-to-peer sync within a local network.

## Tech Stack

- **Flutter**: cross-platform mobile and desktop applications
- **fl_chart**: charts
